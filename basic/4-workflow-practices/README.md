# #4. 워크플로우 실습

이번 섹션에서는 다양한 ComfyUI 워크플로우를 직접 실습해봅니다.

앞선 섹션에서 배운 기본 워크플로우 구성 방법과 LoRA 활용법을 기반으로, 실무에서 자주 사용되는 고급 워크플로우들을 경험해보겠습니다.

## 실습 목록

| 순서 | 주제 | 설명 |
|------|------|------|
| 4-1 | [ControlNet](4-1-controlnet.md) | 포즈, 윤곽선, 깊이 등을 제어하여 원하는 구도의 이미지 생성 |
| 4-2 | [Upscaler](4-2-upscaler.md) | CCSR/SUPIR 기반 고품질 이미지 업스케일링 |
| 4-3 | [Z-Image](4-3-z-image.md) | 알리바바의 Z-Image Turbo 모델로 빠르고 고품질 이미지 생성 |
| 4-4 | [Qwen Image Edit / Layered](4-4-qwen-image.md) | Qwen 기반 이미지 편집 및 레이어 분해 |
| 4-5 | [Image Relighting](4-5-image-relighting.md) | LBM 기반 조명 효과 합성 |
| 4-6 | [Wan 2.2](4-6-wan22.md) | Wan 2.2 모델로 텍스트/이미지 기반 비디오 생성 |
| 4-7 | [Wan 2.2 LoRA](4-7-wan22-lora.md) | Wan 2.2에 LoRA를 적용한 가속 및 스타일 제어 |

## 추천 진행 순서

### 입문자 추천 경로
1. **Z-Image (4-3)** - 가장 간단한 시작점, 빠른 결과 확인
2. **ControlNet (4-1)** - 핵심 기능, 반드시 익혀야 함
3. **Upscaler (4-2)** - 실용적이고 결과가 명확함

### 중급자 추천 경로
4. **Qwen Image Edit (4-4)** - 이미지 편집 기능
5. **Image Relighting (4-5)** - 조명 효과 실습

### 고급자 추천 경로
6. **Wan 2.2 (4-6)** - 비디오 생성, 리소스 많이 필요
7. **Wan 2.2 LoRA (4-7)** - 비디오 생성 가속

## 실습 진행 방식

각 워크플로우 실습은 다음과 같은 구조로 진행됩니다:

1. **개념 소개**: 해당 기술/모델에 대한 간단한 설명
2. **모델 준비**: 필요한 모델 및 커스텀 노드 설치
3. **워크플로우 구성**: 노드 배치 및 연결 (또는 템플릿 로드)
4. **실행 및 결과 확인**: 파라미터 조정 및 결과 비교
